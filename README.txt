Sal Burkolás & Gipszkarton — statikus web sablon
Tartalom:
- index.html
- style.css
- images/ (placeholder képek)

Használat:
1) Csomagold ki a ZIP tartalmát.
2) Töltsd fel a fájlokat egy GitHub repository-ba (Add file → Upload files) vagy Cloudflare Pages-re.
3) GitHub Pages beállítás: Settings -> Pages -> main branch / root. Oldal él: https://yourusername.github.io/repo-nev/
4) Cseréld le az images mappa képeit a saját fotókkal (hero.jpg, thumb1.jpg, full1.jpg, stb).
5) Szükség szerint módosítsd a telefonszámot és e-mail címet az index.html-ben.

Ha szeretnéd, segítek feltölteni GitHub-ra vagy beállítani a domaint.
